# ⏰ GT Smart Clock

**Max7219 tabanlı akıllı dijital saat projesi**

---

## 🚀 Özellikler

- 🌐 Web arayüzü üzerinden metin gönderme  
- 💡 Parlaklık kontrolü  
- 🕒 Saat gösterimi (fallback mekanizması ile)  
- 🎬 Zengin animasyon desteği  
- 📱 Mobil uyumlu modern web arayüzü  

---
